Open job positions

Front-End Developer

Description
Responsible for implementing visual elements that users see and interact with in a web application. Works closely with designers and back-end developers.

Required qualifications
-Proficiency in HTML,CSS, and Javascript
-Familiarity with frameworks like React, Angular, or Vue.js
-Understanding of responsive design principles

Specifications
-Collaborate with UI/UX designers
-Optimize applications for speed and scalability
-Cross-browser compatibility testing

Back-End Developer

Description
Buils and maintains the server-side logic, databases, and APIs that power the web application.

Required qualifications
-Proficiency in server-side languages like Node.js, Python, PHP, or Ruby
-Experience with database systems (MySQL, PostgreSQL, MongoDB)
-Understanding of RESTful APIs and security best practices

Specifications
-Integrate front-end elements with server-side logic
-Write clean, reusable, and efficient code
-Maintain code documentation and testing

UI/UX Designer

Description
Focuses on the design and user experience of the website or web application, ensuring usability and a visually appealing interface.

Required qualifications
-Proficiency in design tools (Figma, Adobe XD, Sketch)
-Strong portfolio of UI/UX design projects
-Knowledge of wireframing, prototyping, and user research

Specifications
-Collaborate with UI/UX designers
-Optimize applications for speed and scalability
-Cross-browser compatibility testing

Web Project Manager

Description
Oversees the planning, execution, and delivery of web development projects. Coordinates teams and ensures deadlines and quality standards are met.

Required qualifications
-Experience with project management tools (Jira, Trello, Asana)
-Strong communication and organizational skills
-Knowledge of web development processes

Specifications
-Create and manage project timelines and budgets
-Act as liaison between clients and development teams
-Ensure projects align with business objectives
